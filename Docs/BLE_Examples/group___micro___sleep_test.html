void aci_hal_fw_error_event ( uint8_t  FW_Error_Type,
uint8_t  Data_Length,
uint8_t  Data[] 
)

This event is generated to report firmware error informations. FW_Error_Type possible values: Values:

  • 0x01: L2CAP recombination failure
  • 0x02: GATT unexpected response
  • 0x03: GATT unexpected request After this event with error type (0x01, 0x02, 0x3) it is recommended to disconnect.
