void DMA_ClearFlag ( uint32_t  DMA_Flag)

Clears the DMAy Channelx's pending flags.

Parameters
DMA_Flagspecifies the flag to clear. This parameter can be any combination (for the same DMA) of the following values:
  • DMA_FLAG_GL1: DMA Channel1 global flag.
  • DMA_FLAG_TC1: DMA Channel1 transfer complete flag.
  • DMA_FLAG_HT1: DMA Channel1 half transfer flag.
  • DMA_FLAG_TE1: DMA Channel1 transfer error flag.
  • DMA_FLAG_GL2: DMA Channel2 global flag.
  • DMA_FLAG_TC2: DMA Channel2 transfer complete flag.
  • DMA_FLAG_HT2: DMA Channel2 half transfer flag.
  • DMA_FLAG_TE2: DMA Channel2 transfer error flag.
  • DMA_FLAG_GL3: DMA Channel3 global flag.
  • DMA_FLAG_TC3: DMA Channel3 transfer complete flag.
  • DMA_FLAG_HT3: DMA Channel3 half transfer flag.
  • DMA_FLAG_TE3: DMA Channel3 transfer error flag.
  • DMA_FLAG_GL4: DMA Channel4 global flag.
  • DMA_FLAG_TC4: DMA Channel4 transfer complete flag.
  • DMA_FLAG_HT4: DMA Channel4 half transfer flag.
  • DMA_FLAG_TE4: DMA Channel4 transfer error flag.
  • DMA_FLAG_GL5: DMA Channel5 global flag.
  • DMA_FLAG_TC5: DMA Channel5 transfer complete flag.
  • DMA_FLAG_HT5: DMA Channel5 half transfer flag.
  • DMA_FLAG_TE5: DMA Channel5 transfer error flag.
  • DMA_FLAG_GL6: DMA Channel6 global flag.
  • DMA_FLAG_TC6: DMA Channel6 transfer complete flag.
  • DMA_FLAG_HT6: DMA Channel6 half transfer flag.
  • DMA_FLAG_TE6: DMA Channel6 transfer error flag.
  • DMA_FLAG_GL7: DMA Channel7 global flag.
  • DMA_FLAG_TC7: DMA Channel7 transfer complete flag.
  • DMA_FLAG_HT7: DMA Channel7 half transfer flag.
  • DMA_FLAG_TE7: DMA Channel7 transfer error flag.
Note
Clearing the Global flag (DMAy_FLAG_GLx) results in clearing all other flags relative to the same channel (Transfer Complete, Half-transfer Complete and Transfer Error flags: DMAy_FLAG_TCx, DMAy_FLAG_HTx and DMAy_FLAG_TEx).
Return values
None
