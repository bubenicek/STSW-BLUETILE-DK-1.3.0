void I2C_ClearITPendingBit ( I2C_Type *  I2Cx,
uint32_t  I2C_IT 
)

Clears the I2Cx interrupt pending bits.

Parameters
I2Cxwhere x can be 1 or 2 to select the I2C peripheral I2C_Type
I2C_ITspecifies the interrupt pending bit to clear. This parameter can be any combination of the following values:
  • I2C_IT_TXFE: Tx FIFO empty
  • I2C_IT_TXFNE: TX FIFO nearly empty
  • I2C_IT_TXFF: Tx FIFO full
  • I2C_IT_TXFOVR: Tx FIFO overflow
  • I2C_IT_RXFE: Rx FIFO empty
  • I2C_IT_RXFNF: Rx FIFO nearly full
  • I2C_IT_RXFF: Rx FIFO full
  • I2C_IT_LBR: Length number of bytes received
  • I2C_IT_RFSR: Read from slave request
  • I2C_IT_RFSE: Read from slave empty
  • I2C_IT_WTSR: Write to slave request
  • I2C_IT_MTD: Master transaction done
  • I2C_IT_STD: Slave transaction done
  • I2C_IT_SAL: Slave arbitration lost
  • I2C_IT_MAL: Master arbitration lost
  • I2C_IT_BERR: Bus error
  • I2C_IT_MTDWS: Master transaction done without stop
  • I2C_IT_TIMEOUT: Timeout or Tlow error
Return values
None
