void SPI_ClearITPendingBit ( uint8_t  SPI_IT)

Clears the SPI interrupt pending bits.

Parameters
SPI_ITspecifies the SPI interrupt pending bit to clear. This parameter can be one of the following values:
  • SPI_IT_TX: Transmit FIFO half empty or less condition interrupt mask.
  • SPI_IT_RX: Receive FIFO half full or less condition interrupt mask.
  • SPI_IT_RT: Receive FIFO not empty and no read prior to timeout period interrupt mask.
  • SPI_IT_ROR: Receive FIFO written to while full condition interrupt mask.
  • SPI_IT_TUR: Transmit underrun interrupt mask.
  • SPI_IT_TE: Transmit FIFO empty interrupt mask.
Return values
None

void SPI_DelayBetweenFrames ( uint8_t  Delay)

Configure the delay between the frames. The MSPIWAIT value is used to insert a wait state between frames. The SSPFRM pulse duration is given by the equation: SSPCLKOUT x {MSPIWAIT - (SCR-1) / [2x(SCR+1)]}. When SCR=1, MSPIWAIT indicates the number of SSPCLKO cycle that the SSPFRM pulse duration should have between frames.

Parameters
Delaythis value must be less than or equal to 15. 0 means no delay (default).
Return values
None
