void SysCtrl_PeripheralClockCmd ( uint32_t  PeriphClock,
FunctionalState  NewState 
)

Enables the clock for the specified peripheral.

Parameters
PeriphClockPeripheral to be enabled, this value can be
  • CLOCK_PERIPH_GPIO Clock of GPIO peripheral
  • CLOCK_PERIPH_NVM Clock of NVM peripheral
  • CLOCK_PERIPH_SYS_CONTROL Clock of SYSTEM CONTROL peripheral
  • CLOCK_PERIPH_UART Clock of UART peripheral
  • CLOCK_PERIPH_SPI Clock of SPI peripheral
  • CLOCK_PERIPH_WDG Clock of WATCHDOG peripheral
  • CLOCK_PERIPH_ADC Clock of ADC peripheral
  • CLOCK_PERIPH_I2C1 Clock of I2C1 peripheral
  • CLOCK_PERIPH_I2C2 Clock of I2C2 peripheral
  • CLOCK_PERIPH_MTFX1 Clock of MFTX1 peripheral
  • CLOCK_PERIPH_MTFX2 Clock of MFTX2 peripheral
  • CLOCK_PERIPH_RTC Clock of RTC peripheral
  • CLOCK_PERIPH_DMA Clock of DMA peripheral
  • CLOCK_PERIPH_RNG Clock of RNG peripheral
  • CLOCK_PERIPH_PKA Clock of PKA peripheral
NewStateNew state of the peripheral clock. This parameter can be: ENABLE or DISABLE.
Return values
None
