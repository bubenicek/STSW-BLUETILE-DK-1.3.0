Detailed Description

This example demonstrate the possibility of multi-function timers configured in mode 3 (dual independent timer/counter).

ATOLLIC project

To use the project with ATOLLIC TrueSTUDIO for ARM, please follow the instructions below:

  1. Open the ATOLLIC TrueSTUDIO for ARM and select File->Import... Project menu.
  2. Select Existing Projects into Workspace.
  3. Select the ATOLLIC project
  4. Select desired configuration to build from Project->Manage Configurations
  5. Select Project->Rebuild Project. This will recompile and link the entire application
  6. To download the binary image, please connect STLink to JTAG connector in your board (if available).
  7. Select Project->Download to download the related binary image.
  8. Alternatively, open the BlueNRG1 Flasher utility and download the built binary image.

KEIL project

To use the project with KEIL uVision 5 for ARM, please follow the instructions below:

  1. Open the KEIL uVision 5 for ARM and select Project->Open Project menu.
  2. Open the KEIL project C:\{username}-1_2 DK x.x.x\Project\BlueNRG1_Periph_Examples\MFT\Mode_1a\MDK-ARM\{BlueNRG-1-2}\MFT_Mode_3.uvprojx
  3. Select desired configuration to build
  4. Select Project->Rebuild all target files. This will recompile and link the entire application
  5. To download the binary image, please connect STLink to JTAG connector in your board (if available).
  6. Select Project->Download to download the related binary image.
  7. Alternatively, open the BlueNRG1 Flasher utility and download the built binary image.

IAR project

To use the project with IAR Embedded Workbench for ARM, please follow the instructions below:

  1. Open the Embedded Workbench for ARM and select File->Open->Workspace menu.
  2. Open the IAR project C:\{username}-1_2 DK x.x.x\Project\BlueNRG1_Periph_Examples\MFT\Mode_1a\EWARM\{BlueNRG-1-2}\MFT_Mode_3.eww
  3. Select desired configuration to build
  4. Select Project->Rebuild All. This will recompile and link the entire application
  5. To download the binary image, please connect STLink to JTAG connector in your board (if available).
  6. Select Project->Download and Debug to download the related binary image.
  7. Alternatively, open the BlueNRG1 Flasher utility and download the built binary image.

Usage

In mode 3 (dual independent timer/counter), the timer/counter is configured to operate as a dual independent system timer or dual external-event counter. In addition, Timer/Counter 1 can generate a 50% duty cycle PWM signal on the TnA pin. The TnB pin can be used as an external-event input or pulse-accumulate input, and serve as the clock source to either Timer/Counter 1 or Timer/Counter 2. Both counters can also be operated from the prescaled system clock.

In this example MFT1 is used. CPU clock is selected as clock signal for Timer/Counter 1. Timer/Counter 2 uses external clock on TnB pin. Sensitivity input to rising edge is selected for TnB. Timer/Counter 1 is preset and reloaded to 5000 so the frequency of the output signal is 100 Hz. The IO2 pin is configured as PWM output with the 100 Hz frequency.

Timer/Counter 2 is preset and reloaded to 5 and its clock source comes from the TnB pin. So, if the IO2 pin is connected to the IO3 pin, providing the clock source of 100 Hz to the Timer/Counter 2, TnA input (MFT1).

The LED DL1 is toggled on Timer/Counter 2 reload (TnD interrupt), so at each 50 ms.
