Detailed Description

This is a demo that shows how to configure BlueNRG-1,2 sleep modes in order to save power. The standalone Virtual/Sleep timer is used for testing the SLEEPMODE_WAKETIMER sleep mode.

ATOLLIC project

To use the project with ATOLLIC TrueSTUDIO for ARM, please follow the instructions below:

  1. Open the ATOLLIC TrueSTUDIO for ARM and select File->Import... Project menu.
  2. Select Existing Projects into Workspace.
  3. Select the ATOLLIC project
  4. Select desired configuration to build from Project->Manage Configurations
  5. Select Project->Rebuild Project. This will recompile and link the entire application
  6. To download the binary image, please connect STLink to JTAG connector in your board (if available).
  7. Select Project->Download to download the related binary image.
  8. Alternatively, open the BlueNRG1 Flasher utility and download the built binary image.

Usage

This test provides an example for the BlueNRG-1,2 sleep modes. The sleep modes tested are SLEEPMODE_WAKETIMER and SLEEPMODE_NOTIMER.

  • The SLEEPMODE_WAKETIMER puts the BlueNRG-1,2 in deep sleep with the timer clock sources running. The wakeup sources are any character typed on the keyboard, the PUSH1 button or the virtual/sleep timer configured with a timeout of 5 sec.
  • The SLEEPMODE_NOTIMER puts the BlurNRG-1,2 in deep sleep with the sleep timer clock sources turned off. The wakeup sources are only any character typed on the keyboard and the PUSH1 button.

The demo supports some user commands:

  • s: SLEEPMODE_NOTIMER - wake on uart/button PUSH1.
  • t: SLEEPMODE_WAKETIMER - wake on uart/timeout 5 sec/button PUSH1.
  • l: Toggle led DL1.
  • p: Print Hello World message.
  • r: Reset the BlueNRG-1.
  • ?: Display this help menu.
  • PUSH1 button: toggle led DL1.
