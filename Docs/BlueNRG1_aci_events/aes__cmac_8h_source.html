26 /*
27 * \page Tutorial_AES_CMAC AES-CMAC Tutorial
28 *
29 * Please remember that before starting to call the Encryption function the context \b requires user
30 * initialization. The Key Size, Tag Size, Flags member must be initialized prior to calling the
31 * init function.
32 * Before the last call to AES_CMAC_Encrypt_Append the flag E_SK_FINAL_APPEND must be set in the context.
33 * Look at the each function documentation to see what is needed prior of calling.
34 *
35 * The API functions to be used are:
36 * - \ref AES_CMAC_Encrypt_Init initialize an \ref AESCMACctx_stt context with key, tag size, flags.
37 * - \ref AES_CMAC_Encrypt_Append process the input. It doesn't produce any output.
38 * It can be called multiple times but the input size must be multiple of 16.
39 * A last append call is allowed with any, positive, input size, but prior to that the flag
40 * E_SK_FINAL_APPEND must be set.
41 * - \ref AES_CMAC_Encrypt_Finish can be called only one time for the finalization process and it will generate
42 * the authentication TAG
43 *
44 * Here follows an example of AES-128-CMAC authentication, it will process a NIST test vector and generate the
45 * authentication TAG.
46 *
47 * \code
48 * #include <stdio.h>
49 * #include "crypto.h"
50 *
51 * int main()
52 * {
53 * const uint8_t KEY_CMAC[] ={0x2b,0x7e,0x15,0x16,0x28,0xae,0xd2,0xa6,0xab,0xf7,0x15,0x88,0x09,0xcf,0x4f,0x3c};

57 * 0xf6,0x9f,0x24,0x45,0xdf,0x4f,0x9b,0x17,0xad,0x2b,0x41,0x7b,0xe6,0x6c,0x37,0x10};
