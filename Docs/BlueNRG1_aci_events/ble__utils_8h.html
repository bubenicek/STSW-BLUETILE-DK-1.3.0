Multiple_Connection_type *  Device_Connection_Params

This API calculates the multiple connections parameters to be used based on the input values.

Parameters
Num_MastersSpecifies numer of master devices to which connects
Num_SlavesSpecifies numer of slave devices to which connects
Scan_WindowSpecifies minimal scan window length requested from user application
Sleep_TimeSpecifies additional time to be added to the minimal anchor period (for optimizig power consumption)
[out]Device_Connection_ParamsSpecifies the connections parameters to be used for handling the multiple connections
Note
Refer to the Multiple_Connections_Formula.html guideline for more information about how formula works
Return values
Valueindicating success or error code.
