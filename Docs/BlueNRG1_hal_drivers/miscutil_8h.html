void HAL_GetCrashInfo ( crash_info_t *  crashInfo)

Get Crash Information utility.

This function return the crash information that are stored in RAM, by hard handler, nmi handler and assert handler. This function reset the crash information stored in RAM before it returns. So it avoid to report the same crash information after a normal reboot.
