What is BlueNRG-1, BlueNRG-2 Radio low level driver ?

  • The BlueNRG-1, BlueNRG-2 devices integrate a RF transceiver compliant to the Bluetooth specification and to the standard national regulations in the unlicensed 2.4 GHz ISM band.
  • The associated Radio low level driver provides access to the BlueNRG-1, BlueNRG-2 2.4 GHz proprietary radio in order to send and receive packets without using the Bluetooth link layer.
  • The 2.4 GHz proprietary radio examples are built on top of the Radio Low level driver and they can be used as reference examples for building other applications that use the BlueNRG-1, BlueNRG-2 2.4 GHz proprietary radio.

First option: use BlueNRG-1,2 Navigator

  • BlueNRG-1,2 Navigator are user friendly GUI which lets you directly download and run the selected 2.4 GHz proprietary radio prebuilt application binary image on the BlueNRG-1,2 platforms without a SWD interface (it just requires a BlueNRG-1, STEVAL-IDB007V2 development kit, or a BlueNRG-2, STEVAL-IDB008V1, BlueNRG-2, STEVAL-IDB008V2, BlueNRG-248, STEVAL-IDB009V1 connected to a PC USB port)
  • They also provide demo applications description (html files) and access to board configurations and demo application source code if needed.
  • BlueNRG-1,2 Navigator are available on BlueNRG-1_2 DK SW package on Start menu folder, ST BlueNRG-1_2 DK X.X.X, BlueNRG-1 Navigator or BlueNRG-2 Navigator.
  • User is just request to connect one or two BlueNRG-1, STEVAL-IDB007V2 development kit, or a BlueNRG-2, STEVAL-IDB008V1, BlueNRG-2, STEVAL-IDB008V2, BlueNRG-248, STEVAL-IDB009V1board on PC USB ports and then select the specific 2.4 GHz proprietary radio demonstration application to be tested.

  • Refer to the BlueNRG-1,2 development kit User Manual (UM2071), BlueNRG-1,2 Navigator section for more detailed information.

Second option: move to SW development environment starting from available 2.4 GHz proprietary radio examples

  • BlueNRG-1_2 DK SW package provides a set of 2.4 GHz proprietary radio examples allowing to test several applications scenarios which can be easily selected and downloaded on a BlueNRG-1, STEVAL-IDB007V2 development kit, or a BlueNRG-2, STEVAL-IDB008V1, BlueNRG-2, STEVAL-IDB008V2, BlueNRG-248, STEVAL-IDB009V1 platform using the BlueNRG-1,2 Navigator tools.
  • User can identify which available demonstration application is close to his specific needs and follows such steps:
    • Open its IDE toolchains and related demonstration application project (IAR EWARM for ARM, Keil MDK-ARM, Atollic TrueStudio for ARM toolchains examples are available on BlueNRG-1_2 DK SW package )
    • Modify the available source files according to his needs and then download and test the built binary file on the selected platform.
    • Run the built application and play with it!
