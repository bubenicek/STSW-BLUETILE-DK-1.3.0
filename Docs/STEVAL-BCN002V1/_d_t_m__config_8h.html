◆ DTM_NUM_LINK

#define DTM_NUM_LINK   8

This file contains all the information needed to init the BlueNRG-1 stack. These constants and variables are used from the BlueNRG-1,2 stack to reserve RAM and FLASH according the application requests. NOTE: the current selected configuration is tuned for supporting up to 8 connections, 8 services, 68 attributes. MAX ATT_MTU size is set to 220 bytes which is the max allowed value for fitting with the overall DTM RAM memory requirements for all the supported configurations (SPI and UARTs), and use the extended packey length for OTA client FW upgrade procedure. This implies to reduce the PREPARE_WRITE_LIST_SIZE value. User can modify, tune the configuration options according to his specific application requirements.
