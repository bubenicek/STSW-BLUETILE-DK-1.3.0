◆ hidSetIdleTimeout()

void hidSetIdleTimeout ( uint32_t  timeout)

Sets the Idle Connection Timeout according the HID over GATT profile specification. The HID device may perform the GAP Terminate Connection procedure if the connection is idle for a time period, which is implementation dependent. After bonding the HID device should wait for this idle connection timeout to allow the HID Host to complete the configuration procedure. Default value 500 ms.

Note
If the timeout is zero the Power Save management is disabled
Parameters
timeoutIdle connection timeout expressed in number of millisecond.
Return values
Nothing
